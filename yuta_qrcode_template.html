現場操作步驟

  1. 各組平板掃描左側 QR Code，直接進入選組頁面
  2. 每組選擇自己的組別，按「召喚猶他大師」
  3. 老師裝置掃描右側 QR Code，進入監控儀表板（需輸入通關密語）
  4. 工作坊開始前，老師可在儀表板按「清除所有對話記錄」重置

⚙️ 使用前設定（列印前請先完成，此區塊列印時不顯示）

  1. 將 YOUR_BASE64_ENCODED_API_KEY 替換為你的 Anthropic API key 的 base64 編碼字串
  2. 將 YOUR_GITHUB_USERNAME 替換為你的 GitHub 帳號名稱
  3. 確認替換完成後，將此檔案存為 yuta_qrcode.html（注意：不要上傳至 GitHub）
  4. 在瀏覽器開啟確認 QR code 正確生成後，列印 A4 備用
  5. 列印完成後刪除本機的 yuta_qrcode.html，僅保留此範本
